Default Configuration adalah konfigurasi bawaan yang sudah tersedia pada router MikroTik setelah di reset atau pertama kali digunakan.

Langkah-langkah:

1). Login ke MikroTik menggunakan WinBox.

2). Pilih menu System-Reset Configuration.

<img width="400" height="250" alt="cpyan " src="https://github.com/user-attachments/assets/ac1357bc-eca1-416f-bcc8-32b0bcd5f81e" />

3). Lalu klik Reset Configuration.

<img width="400" height="250" alt="Screenshot (115)" src="https://github.com/user-attachments/assets/b46dbfc1-4ea4-4e19-acf4-14b584bb6752" />

4). Setelah selesai melakukan Reset Conguration, login kembali menggunakan WinBox.

5) Login dengan IP 182.168.88.1, user "admin", password kosongkan, lalu klik connect.

<img width="400" height="250" alt="Screenshot (118)" src="https://github.com/user-attachments/assets/16477220-663d-43f7-8331-971b00afa5bd" />

Default Configuration berisi:

a. IP Address 192.168.88.1/24.

b. Username "admin", password kosong.

c. DHCP Server aktif pada jaringan 192.168.88.0/24.

d. NAT untuk akses ke internet.

e. DNS untuk menerjemahkan nama domain menjadi alamat IP.
