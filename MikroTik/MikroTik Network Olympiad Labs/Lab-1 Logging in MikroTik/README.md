Ada beberapa cara untuk login ke perangkat Mikrotik, yaitu:
1. Login Menggunakan WinBox
Winbox adalah aplikasi yang digunakan untuk mengelola mikrotik melalui GUI (Graphical User Interface).Pengguna dapat login menggunakan MAC Address atau IP Address router.
Download WinBox [WinBox_Windows (1).zip](https://github.com/user-attachments/files/29338716/WinBox_Windows.1.zip)
Langkah-langkah:
1). Hubungkan komputer ke router MikroTik menggunakan kabel LAN.
2). Buka aplikasi WinBox.
   <img width="150" height="50" alt="1745584261358" src="https://github.com/user-attachments/assets/15f513f3-7d3b-41b9-9413-457f796c318a" />
<img width="350 height="200" alt="Screenshot (149)" src="https://github.com/user-attachments/assets/288d9359-4e84-425e-ae1c-e39eb305d6a3" />
3). Klik MAC Address atau klik IP Address.
4). Login "admin" Password kosongkan, lalu klik connect.
<img width="350" height="200" alt="Screenshot (113)" src="https://github.com/user-attachments/assets/2b1ce1b6-4879-4f8c-9a79-2994d27e00c3" />

2. Login Menggunakan PuTTy
PuTTy digunakan untuk mengakses MikroTik melalui Command Line Interface (CLI) menggunakan protokol SSH.Metode ini cocok untuk konfigurasi menggunakan perintah-perintah terminal.

1). Pastikan komputer dan router MikroTik saling terhubung.
2). Buka Aplikasi PuTTy
<img width="150" height="50" alt="image" src="https://github.com/user-attachments/assets/e9a81c00-7bb8-4d64-aa50-eab0f1053c3c" />
3). Ketik IP Address 192.168.88.1 (IP default MikroTik).
4). Ketik port 23 (port default Telnet).
5). Lalu klik open.

6). Login "admin" Password kosongkan, lalu klik enter.

3. Login Menggunakan WebFig
WebFig adalah utilitas berbasis web milik MikroTik yang berfungsi untuk memantau, dan mengonfigurasi pada router MikroTik.
Langkah-langkah:
1). Pastikan komputer dan router MikroTik saling terhubung.
2). Buka Aplikasi Chrome
