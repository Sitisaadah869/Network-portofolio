Command Line Interface (CLI) adalah antramuka berbasis perintah teks yang digunakan untuk mengonfigurasi dan mengelola router MikroTik melalui Terminal.Selain menggunakan GUI MikroTik juga dapat dikonfigurasi menggunakan mode CLI (berbasis teks/text mode).

Langkah-langkah:

1). Pastikan komputer dan router MikroTik saling terhubung.
2). Buka WinBox, login menggunakan user "admin", password kosongkan.
3). Klik New Terminal.
